JavaScript adalah bahasa pemrogramman tingkat tinggi yang dikembangkan untuk membuat halaman web menjadi lebih interaktif. JavaScript memungkinkan menambahkan logika pemrogrammman, seperti menangani interaksi pengguna, memanipulasi data, hingga melakukan request ke server.


Tipe data : 
String, tipe data yang digunakan untuk menyimpan dan memanipulasi teks 
Number, tipe data yang menyimpan angka, termasuk bilangan bulat dan desimal
Boolean, tipe data yang memiliki nilai true atau false 
Undefined, tipe data yang memiliki variabel dan nilai, namun belum didefinisikan
Null, tipe data yang memiliki variabel, tapi tidak mempunyai nilai


Terdapat 3 Jenis Operator :
a. Operator Aritmatika
- Penjumlahan +
- Pengurangan -
- Perkalian *
- Pembagian /
- Modulus %
- Eksponen **
- Increment ++
- Decrement --

b. Operator Perbandingan
- Membandingkan persamaan nilai ==
- Membandingkan persaaan nilai dan tipe data ===
- Membandingkan ketidaksamaan nilai dari vaiabel !=
- Membandingkan pertidaksamaan nilai dan tipe data !==
- Lebih besar > 
- Membandingkan apakah nilainya sama atau lebih besar >=
- Lebih kecil <
- Membandingkan apakah nilainya sama atau lebih kecil <=

c. Operator Logika 
- And &&
- Or ||
- Not !


If-Else Statement digunakan untuk membuat keputusan berdasarkan suatu kondisi. Jika kondisinya true, blok tertentu akan dijalankan, dan jika salah, kode lain akan dijalankan.

Switch-Case adalah pernyataan suatu kondisi yang digunakan untuk membandingkan beberapa kemungkinan nilai dan mengeksekusi blok kode tertentu berdasarkan kecocokan nilai tersebut. 

⁢Function ⁡⁣⁣⁢adalah blok kode yang dibuat untuk melakukan tugas tertentu dan dapat digunakan berulang kali.

Parameter adalah variabel yang didefinisikan dalam deklarasi function.

Return : Mengembalikan nilai dari console. Setela return dieksekusi, ekskusi function akan terhenti
