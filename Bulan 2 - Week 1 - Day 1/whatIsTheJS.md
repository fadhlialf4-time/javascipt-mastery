JavaScript adalah bahasa pemrogramman tingkat tinggi yang dikembangkan untuk membuat halaman web menjadi lebih interaktif.

JavaScript memungkinkan menambahkan logika pemrogrammman, seperti menangani interaksi pengguna, memanipulasi data, hingga melakukan request ke server.

Object Oriented Programming (OOP) adalah gaya atau paradigma pemrogramman yang berfokus pada objek sebagai cara untuk mengatur kode dan data
